To build the JavaScript code in a single, compressed file run the
following commands in the current directory:

$ jsbuild -o ../africamap/public/build/mapfish app.cfg
$ cp -r ../africamap/public/mfbase/mapfish/img ../africamap/public/build/mapfish/
$ cp -r ../africamap/public/mfbase/openlayers/img ../africamap/public/build/openlayers/
$ cp -r ../africamap/public/mfbase/openlayers/theme ../africamap/public/build/openlayers/
